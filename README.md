DEEBOY.COM Online Shop Demo
DEEBOY.COM is a simple online shop demo built with HTML, CSS, and JavaScript.

It is designed for GitHub Pages, so it can be published as a static website without PHP or MySQL.

Features
Product display cards
Product images
Add to cart buttons
Cart item counter
Cart list
Total price calculation
Responsive layout
